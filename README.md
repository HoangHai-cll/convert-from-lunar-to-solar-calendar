BMI Calculator với Chuyển đổi Lịch Âm Dương
Mô tả dự án
Đây là một ứng dụng Android đơn giản được phát triển để tính toán ,cung cấp chức năng chuyển đổi ngày tháng từ lịch Dương lịch (Gregorian) sang lịch Âm lịch Việt Nam. Ứng dụng sử dụng các thuật toán thiên văn học để tính toán chính xác ngày âm lịch, bao gồm cả năm nhuận và tháng nhuận.
Tính năng chính
•
Chuyển đổi lịch: Chuyển đổi ngày Dương lịch sang ngày Âm lịch, bao gồm:
◦
Ngày, tháng, năm âm lịch.
◦
Xác định năm nhuận (nếu có).
◦
Hiển thị năm theo Can Chi (ví dụ: Giáp Thân, Ất Dậu, v.v.).
Cách sử dụng
1.
Mở ứng dụng trên thiết bị Android.
2.
Nhập ngày, tháng, năm Dương lịch.
3.
Ứng dụng sẽ hiển thị ngày tương ứng trong lịch Âm lịch Việt Nam.
Cấu trúc dự án
•
LunarCalendar.java: Lớp chính chứa logic chuyển đổi lịch âm dương, dựa trên thuật toán thiên văn học chuẩn.
•
Các file Gradle: Cấu hình build cho Android.
Yêu cầu hệ thống
•
Android API 21 trở lên.
•
Java 8 hoặc cao hơn.
Cài đặt và chạy
1.
Clone repository này.
2.
Mở trong Android Studio.
3.
Build và chạy trên emulator hoặc thiết bị thật.
